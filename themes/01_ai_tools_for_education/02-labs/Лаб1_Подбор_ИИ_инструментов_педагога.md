# ЧАСТЬ I. ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ

## 1.1. Критерии выбора ИИ-инструментов

При подборе инструментов искусственного интеллекта для профессиональной деятельности педагога необходимо учитывать следующие критерии:

| Критерий | Описание | Приоритет |
|----------|----------|-----------|
| **Доступность в РФ** | работа без VPN, регистрация с российским номером, оплата в рублях | высокий |
| **Русский язык** | качественная поддержка русского языка | высокий |
| **Безопасность данных** | соответствие 152-ФЗ, локальное хранение | высокий |
| **Стоимость** | бесплатный доступ или минимальная стоимость | средний |
| **Наличие ИИ** | инструмент должен использовать AI или легко интегрироваться с AI | высокий |
| **Возможность локальной работы** | работа без интернета (для open-source) | дополнительный |

### 3.2. Составление ручных цепочек инструментов (без полной автоматизации)

Составьте **цепочку (workflow)** из нескольких инструментов для комплексной работы над сложной задачей. Каждый этап выполняется вручную с применением отдельных ИИ-инструментов.

**Пример 1: создание видеоурока с субтитрами**

```
Этап 1: генерация плана урока
   Инструмент: GigaChat / DeepSeek
   Действие: ввести промпт с темой и целями урока
   Результат: структура урока (копируем в документ)
       ↓
Этап 2: создание текста лекции
   Инструмент: GigaChat / Qwen  
   Действие: расписать каждый пункт плана
   Результат: текст для озвучивания
       ↓
Этап 3: генерация иллюстраций
   Инструмент: Kandinsky / Шедеврум / Krita + AI Diffusion (локально)
   Действие: создать изображения по описанию
   Результат: PNG-файлы
       ↓
Этап 4: запись видео/аудио
   Инструмент: OBS Studio / любой рекордер
   Действие: записать экран с презентацией и голос
   Результат: видеофайл
       ↓
Этап 5: транскрибация и субтитры
   Инструмент: Whisper (локально) / Писец / Speech2Text.ru
   Действие: загрузить видео, получить SRT-файл
   Результат: видеоурок с субтитрами
```

**Пример 2: создание комплекта тестовых заданий**

```
Этап 1: анализ учебного материала
   Инструмент: GigaChat Deep Research
   Действие: загрузить учебник/конспект, попросить выделить ключевые темы
   Результат: список тем для проверки
       ↓
Этап 2: генерация вопросов
   Инструмент: GigaChat / DeepSeek
   Действие: для каждой темы сгенерировать 5-10 вопросов разного типа
   Результат: текстовый файл с вопросами и ответами
       ↓
Этап 3: создание визуальных заданий (опционально)
   Инструмент: Kandinsky / Stable Diffusion (локально)
   Действие: сгенерировать изображения для заданий на соотнесение
   Результат: изображения
       ↓
Этап 4: оформление теста
   Инструмент: Online Test Pad / iSpring / Google Forms
   Действие: вручную перенести вопросы в конструктор
   Результат: готовый тест для учащихся
```

### Krita + AI Diffusion

**GitHub:** [github.com/Acly/krita-ai-diffusion](https://github.com/Acly/krita-ai-diffusion)

Плагин для графического редактора Krita:
- интеграция Stable Diffusion прямо в редактор
- inpaint (дорисовка выделенной области)
- outpaint (расширение изображения)

**Установка:**
1. установить Krita 5.2.0+
2. скачать плагин с GitHub
3. Tools → Scripts → Import Python Plugin from File
4. в докере плагина нажать Configure для установки локального сервера

### Inkscape MCP Server

**GitHub:** [github.com/Shriinivas/inkmcp](https://github.com/Shriinivas/inkmcp)

MCP-сервер для управления Inkscape через ИИ-ассистентов (Claude, Gemini и др.):
- создание SVG-элементов через естественный язык
- выполнение Python/inkex кода в живом контексте
- экспорт скриншотов viewport
- иерархическое управление элементами

**Платформа:** только Linux (использует D-Bus)

**Установка:**
```bash
# скачать расширение
cd ~/.config/inkscape/extensions/
unzip inkmcp-extension.zip

# сделать скрипты исполняемыми
cd inkmcp
chmod +x run_inkscape_mcp.sh inkmcpcli.py inkscape_mcp_server.py
```

**Примеры команд для ИИ-ассистента:**
```
"В Inkscape нарисуй синусоиду от левого края документа"
"В Inkscape создай логотип с радиальным градиентом и типографикой"
"В Inkscape нарисуй диаграмму с подписями"
```

### GIMP MCP Server

**GitHub:** [github.com/maorcc/gimp-mcp](https://github.com/maorcc/gimp-mcp)

MCP-сервер для управления GIMP 3.0 через ИИ-ассистентов:
- полный доступ к GIMP API через PyGObject
- редактирование изображений через естественный язык
- экспорт изображений в PNG, JPEG, BMP, TIFF
- ИИ может видеть и анализировать изображения в GIMP

**Платформа:** Windows, macOS, Linux (кросс-платформенный)

**Установка (Linux):**
```bash
git clone https://github.com/maorcc/gimp-mcp.git
cd gimp-mcp
uv sync

# установка плагина GIMP
mkdir -p ~/.config/GIMP/3.0/plug-ins/gimp-mcp-plugin
cp gimp-mcp-plugin.py ~/.config/GIMP/3.0/plug-ins/gimp-mcp-plugin/
chmod +x ~/.config/GIMP/3.0/plug-ins/gimp-mcp-plugin/gimp-mcp-plugin.py
```

**Запуск:**
1. открыть изображение в GIMP
2. Tools → Start MCP Server
3. подключить ИИ-клиент (Claude Desktop, Gemini CLI и др.)

**Примеры команд:**
```
"Осветли фон и добавь винтажный фильтр"
"Убери красные глаза и увеличь резкость"
"Нарисуй лицо и овечку в GIMP"
```

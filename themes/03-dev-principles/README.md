# Материалы темы `03-dev-principles`

## Структура
- `00_teacher_guide.md` - внутренний сценарий преподавателя с таймингом занятия.
- `01_lecture_context_and_es.md` - контекст, определения, характеристики ЭС, классификации ИСО/ЭСО.
- `02_lecture_lifecycle.md` - жизненный цикл ЭСО и критерии качества/полезности.
- `03_lecture_platform_stack.md` - практический стек интеграции и технические ограничения.
- `04_practice_track_A_autotesting.md` - основной практикум: проектирование автотестирования.
- `05_practice_track_B_variant.md` - вариативный простой проект по тому же циклу (`B-lite`).
- `06_debrief_rubric.md` - двухконтурная оценка (`A-core` + обязательный минимум `B-lite`).
- `images/` - исходные и конвертированные схемы.

## Примечание
Во всех студенческих материалах не указываются временные интервалы. Тайминг хранится только в `00_teacher_guide.md`.

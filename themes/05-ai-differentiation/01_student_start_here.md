# 01. С чего начать

## В одной фразе
Вы проводите входную проверку подготовки, собираете мини-модуль по теме `Массивы и циклы` в трех уровнях сложности и подтверждаете качество через сравнение двух контуров: локальная модель и российская облачная модель.

## Перед стартом
1. Прочитайте словарь терминов в [README.md](README.md).
2. Подготовьте инструменты:
   - локальная модель: `Ollama` или `LM Studio`;
   - российская облачная модель: `GigaChat` или `YandexGPT`.

## Порядок выполнения
1. Проведите входную проверку подготовки в [02_lecture_differentiation_foundations.md](02_lecture_differentiation_foundations.md).
2. Постройте трехуровневую матрицу заданий по [03_lecture_three_level_design.md](03_lecture_three_level_design.md).
3. Выполните обязательную часть A (`A-core`) по [05_practice_track_A_differentiated_module.md](05_practice_track_A_differentiated_module.md).
4. Выполните один вариант вариативной части B (`B-variant`) по [06_practice_track_B_variant.md](06_practice_track_B_variant.md).
5. Проверьте итог по рубрике [07_debrief_rubric.md](07_debrief_rubric.md).

## Какой порядок проверки выбрать
- **Базовый порядок проверки в чате (Manual Chat Workflow)**: обязателен, достаточен для зачета.
- **Расширенный автоматизированный порядок проверки (Automation Workflow)**: опционален, выполняется на связке `Yandex Forms -> Albato -> GigaChat -> Google Sheets`.

Если вы не выбираете автоматизацию, это не снижает итог при полном выполнении базового порядка проверки в чате.

## Что нужно сдать
1. Таблицу входной проверки и выбранного уровня.
2. Матрицу: `уровень -> цель -> задание -> ожидаемый ответ -> критерий`.
3. Журнал итераций минимум из двух шагов для каждого контура.
4. Таблицу сравнения двух контуров по пяти критериям.
5. Решение по каждому контуру: принять / доработать / не принимать.
6. Финальный комплект материалов для занятия.

## Быстрая самопроверка
1. Понятно ли, какой шаг вы делаете первым и зачем?
2. Зафиксированы ли результаты после каждого шага, а не только в конце?
3. Есть ли сравнение именно двух контуров, а не одного?

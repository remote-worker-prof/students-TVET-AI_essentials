# 05. Практика: обязательная часть A (A-core)

---

## 🎯 Цель практики
Вы создаете и проверяете **три полноценных учебных комплекта** по одной цели:
- `basic`,
- `standard`,
- `advanced`.

Каждый комплект должен включать:
1. теорию,
2. практику,
3. тест,
4. правильные ответы теста и проходной балл,
5. ошибку для разбора и образец корректного исправления.

`Ошибка для разбора (в учебном примере)` - это элемент учебного материала, а не статистика ошибок группы.

В пакетах собраны:
1. быстрый запуск части A;
2. входная проверка с алгоритмом назначения уровня;
3. чистовые шаблоны Таблиц 1-5;
4. протокол итераций и сравнения контуров;
5. полностью заполненный сквозной пример отчета A.

## 🧱 Исходная база
### 🧱 Опорный фрагмент
1. Массив хранит упорядоченный набор элементов одного типа.
2. Элемент массива выбирается по индексу.
3. Цикл позволяет повторять блок команд.
4. `for` используют, когда известно число повторений.
5. `while` используют, когда повторение зависит от условия.
6. Ошибка для разбора (в учебном примере): выход за границы массива при неверном индексе.
7. При проходе по массиву важно контролировать начальное и конечное значение индекса.

### 🎯 Единая цель
Корректно выбирать цикл для прохода по массиву и обнаруживать ошибку индексации.

### ✅ Единые критерии
`accuracy`, `goal_fit`, `level_fit`, `checkability`, `clarity`.

# 07. Рубрика оценивания и формат защиты

---

## 🎯 Как читать рубрику
Каждый критерий оценивается по шкале `0-2`:
- `0` — критерий не выполнен,
- `1` — выполнен частично,
- `2` — выполнен полностью и подтвержден фактами.

Максимальная сумма по рубрике: `22` балла.

---

## ✅ Рубрика (0-2 балла по каждому критерию)

| Критерий | 0 | 1 | 2 |
|---|---|---|---|
| Таблица 1: стартовый уровень и матрица уровней | отсутствует | заполнена частично | заполнена полностью и логично |
| Таблица 3: полный комплект `basic` | отсутствует | есть неполные блоки | есть теория, практика, тест, правильные ответы теста, проходной балл, ошибка для разбора и образец корректного исправления |
| Таблица 4: полный комплект `standard` | отсутствует | есть неполные блоки | есть теория, практика, тест, правильные ответы теста, проходной балл, ошибка для разбора и образец корректного исправления |
| Таблица 5: полный комплект `advanced` | отсутствует | нет переноса или анализа ошибки | комплект полный, перенос и анализ ошибки присутствуют |
| Таблица 2: сравнение двух контуров | отсутствует | баллы без фактов | баллы подтверждены фактами |
| Решение по качеству в Таблице 2 | отсутствует | формальное | обоснованное и связанное с фактами |
| Разведение режимов шага (`🧱 Разработка материала` / `📑 Отчётность`) | роли смешаны | есть частичные пересечения | роли разделены однозначно и последовательно |
| Два контура моделей | использован один | второй подтвержден частично | оба контура подтверждены полностью |
| Финальный комплект для занятия | не готов к применению | есть недочеты в структуре | комплект применим без дополнительных пояснений |
| Корректность выбранного порядка работы | не подтвержден | подтвержден частично | воспроизводим полностью |
| Результат части B | отсутствует | дублирует данные A | самостоятельная доработка без дублирования A |

## 🎯 Минимум для зачета
- Рекомендуемый минимум: `17` из `22`.
- Обязательные условия:
  1. критерий «Таблица 3: полный комплект `basic`» = `2`;
  2. критерий «Таблица 4: полный комплект `standard`» = `2`;
  3. критерий «Таблица 5: полный комплект `advanced`» = `2`;
  4. критерий «Таблица 2: сравнение двух контуров» = `2`.

---

## 📝 Чек-лист быстрой проверки

| Пункт | Статус |
|---|---|
| Таблицы 1-5 заполнены полностью | `да/нет` |
| Во всех трех уровнях есть теория, практика, тест, правильные ответы теста и проходной балл | `да/нет` |
| В уровне `advanced` есть перенос и анализ ошибки | `да/нет` |
| Таблица 2 содержит факты по двум контурам | `да/нет` |
| Режимы `🧱 Разработка материала` и `📑 Отчётность` не смешаны | `да/нет` |
| Результат части B не дублирует часть A | `да/нет` |

---

## 🧭 Формат защиты
1. Покажите Таблицу 1 и логику назначения стартового уровня.
2. Покажите Таблицы 3-5 как полные комплекты уровней.
3. Покажите Таблицу 2 и обоснование итогового решения.
4. Покажите финальный комплект и результат части B.

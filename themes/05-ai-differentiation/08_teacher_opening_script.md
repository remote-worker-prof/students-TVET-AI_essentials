# 08. Вводный сценарий запуска темы (внутренний файл)

---

## 🎯 Текст открытия (2-4 минуты)
Сегодня вы запускаете тему, где итогом являются не отдельные ответы, а три полноценных учебных комплекта:
- `basic`,
- `standard`,
- `advanced`.

В каждом уровне обязательно должны быть:
1. теория,
2. практика,
3. тест,
4. правильные ответы теста и проходной балл,
5. ошибка для разбора и образец корректного исправления.

`Ошибка для разбора (в учебном примере)` - это часть учебного материала, а не статистика ошибок группы.

---

## 🧭 Порядок действий на старте
1. Выполняется входная проверка и назначается стартовый уровень.
2. Заполняется Таблица 1 и собираются полные комплекты уровней (Таблицы 3-5).
3. Выполняется сравнение двух контуров в Таблице 2.
4. Принимается решение по качеству: принять / доработать / не принимать.
5. Выполняется один вариант части B без дублирования таблиц части A.

---

## 🧪 Два маршрута проверки
- Базовый порядок проверки в чате обязателен.
- Автоматизированный маршрут `Yandex Forms -> Albato -> GigaChat -> Google Sheets` опционален.
